# Crawl Scaffolding

This folder is a temporary helper for AI track crawl activities.

## Usage
- Use chain-PRs: keep changes small and incremental per step.
- Include evidence in PRs: commands run, outputs summarized, and files changed.
- Keep prompts used for each step so runs are reproducible.

## Lifecycle
- Keep during the track.
- Discard this scaffolding after track completion.
